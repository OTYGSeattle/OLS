OLS v1.0 supports fine-grained verse structure for Psalms, Deggwa hymns, and antiphonal performance.


```json
{
  "type": "hemistich-pair",
  "n": 1,
  "lines": [
    {
      "n": "1a",
      "hemistich": "a",
      "roles": ["role-cantor"],
      "text": { "en": "The Lord is my shepherd" }
    },
    {
      "n": "1b",
      "hemistich": "b",
      "roles": ["role-congregation"],
      "text": { "en": "I shall not want" }
    }
  ]
}
```
