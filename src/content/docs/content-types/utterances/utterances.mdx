An `Utterance` is the atomic unit of liturgical speech: spoken, chanted, sung, whispered, silent, or interior.


```json
{
  "id": "ut-kyrie-deacon",
  "roles": ["role-deacon"],
  "mode": "chanted",
  "delivery": "solemn",
  "audience": "divine",
  "text": {
    "gez-Ethi": "ኪርያ ኤሌይሶን",
    "gez-Latn": "Kyrie Eleison",
    "en": "Lord, have mercy"
  },
  "chant": {
    "system": "yared",
    "mode": "geez"
  }
}
```
