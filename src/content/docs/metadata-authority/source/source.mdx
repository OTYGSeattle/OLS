OLS v1.0 requires serious liturgical content to identify its source. This protects against confusion between official texts, study translations, parish handouts, and drafts.

## Citation

```json
{
  "citation": {
    "id": "src-mesHafe-qeddase-ed1-p42",
    "type": "printed-book",
    "title": "Mäṣḥafä Qeddase",
    "language": "gez",
    "edition": "unknown",
    "publisher": null,
    "year": null,
    "page": "42",
    "line": "3",
    "license": "restricted",
    "notes": {
      "en": "Exact edition requires verification."
    }
  }
}
```

## Provenance

```json
{
  "provenance": {
    "sourceStatus": "transcribed",
    "transcriber": "person-transcriber-001",
    "translator": "person-translator-001",
    "editors": ["person-editor-001"],
    "reviewers": ["person-priest-001", "person-geez-scholar-001"],
    "confidence": "medium",
    "lastReviewed": "2026-06-29",
    "changeReason": "Initial v1.0 transcription from printed source."
  }
}
```

## Citation and Provenance Work Together

A citation identifies the evidence; provenance records what happened between that evidence and the encoded entity. Keep them separate so a correction to a transcription does not rewrite the source record.

```json
{
  "source": "src-mesHafe-qeddase-ed1-p42",
  "provenance": {
    "sourceStatus": "transcribed",
    "transcriber": "person-transcriber-001",
    "translator": "person-translator-001",
    "confidence": "medium",
    "lastReviewed": "2026-06-29"
  }
}
```

Use `null` for an explicitly unknown bibliographic value and omit a field only when it is not applicable or not collected. Notes such as “exact edition requires verification” are useful, but they do not substitute for structured edition, page, line, license, and confidence fields.

For recordings and field notes, identify the recording date, location, participants or privacy constraints in the applicable source metadata. A performer’s rendition is evidence of a practice; it should not automatically be labeled an official text.
