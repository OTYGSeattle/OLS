---
title: 'Execution Timeline Binding'
description: 'Timeline binding of real performance timestamps, durations, and status.'
category: 'space-actions'
order: 29
updatedAt: 2026-07-01
---


Actual performance timestamps can bind to Sections, Blocks, Readings, Rubrics, and Utterances.


```json
{
  "executionTimeline": [
    {
      "target": "sec-anaphora",
      "type": "section",
      "start": "08:12:30",
      "end": "09:05:10",
      "duration": "PT52M40S",
      "status": "completed"
    }
  ]
}
```
